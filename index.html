HTML
要素解説

<p>要素

説明

<p>要素はテキストの段落を表します。 文章上で段落は字下げなどをして表しますが、 HTMLでは段落を<p>要素でテキストを囲むことによって1つのコンテンツとして扱うことが出来ます。

使用例

                            <!DOCTYPE html>
                            <html lang="ja">
                                <head>
                                    <meta charset="utf-8">
                                    <title>Example</title>
                                </head>
                                <main>
                            
                                    <p>Hello world!</p>
                                    
                                </main>
                            </html>

出力

Hello world!

strong要素

説明

strong要素はテキストを太字にします。 主にテキストを強調したいときに使います。

使用例

                            <!DOCTYPE html>
                            <html lang="ja">
                                <head>
                                    <meta charset="utf-8">
                                    <title>Example</title>
                                </head>
                                <main>
                            
                                    <p>Hello <strong>happy</strong> world!</p>
                                    
                                </main>
                            </html>

出力

Hello happy world!

h1要素

説明

<h1>要素はテキストを見出しにしたいときに使います。 <h1>要素に似た要素に<h2>～<h6>要素があります。 これらの要素は数字が小さいほど大きな見出しとなるので<h1>要素は主題などの最も大きい見出しで使われます。 <h1>要素はwebページに1回だけ使われるのが理想です。

使用例

                            <!DOCTYPE html>
                            <html lang="ja">
                                <head>
                                    <meta charset="utf-8">
                                    <title>Example</title>
                                </head>
                                <main>

                                    <h1>Hello world!</h1>
                                    <h2>Hello world!</h2>
                                    <h3>Hello world!</h3>
                                    <h4>Hello world!</h4>
                                    <h5>Hello world!</h5>
                                    <h6>Hello world!</h6>
                                    
                                </main>
                            </html>

出力

                            
Hello world!

                            
Hello world!

                            
Hello world!

                            
Hello world!

                            
Hello world!

                            
Hello world!
